# MineLauncher

A custom **Minecraft launcher** built with a focus on simplicity, customization, and a clean modern interface.
MineLauncher allows players to easily launch Minecraft with **Fabric modpacks**, manage installations, and log in using a **real Microsoft account**.

This project was created as a custom launcher experiment combining modern UI design with practical modded Minecraft support.

---

## Features

* Microsoft account authentication
* Fabric mod loader support
* Modrinth modpack compatibility
* Instance / installation manager
* Clean modern launcher interface
* Built-in logging system
* Easy launch for modded Minecraft versions

---

## Installation

1. Go to the **Releases** page of this repository.
2. Download the latest `.exe` build.
3. Run the launcher and log in with your Microsoft account.
4. Select or install an instance and launch Minecraft.

Download here:
